# EndPoints:

#### //USERS  
GET /api/users/  
POST /api/users/  
PUT /api/users/{id}  
DELETE /api/users/{id}  

#### //GAMES  
GET /api/games/  :OK:  
POST /api/games/  
PUT /api/games/{id}  
DELETE /api/games/{id}  

#### //PARTIES
GET /api/parties/  
POST /api/parties/  
PUT /api/parties/{id}  
DELETE /api/parties/{id}  

#### //MESSAGES
GET /api/messages/  
POST /api/messages/  
PUT /api/messages/{id}  
DELETE /api/messages/{id}
